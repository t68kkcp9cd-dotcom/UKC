## Getting Started

### Creating Your Account

1. **Web Registration**
   - Navigate to your UKC instance
   - Click "Sign Up" on the login page
   - Enter username, email, and password
   - Optionally join an existing household
   - Click "Create Account"

2. **Mobile App Registration**
   - Open the UKC app
   - Tap "Sign Up"
   - Enter server URL (e.g., http://your-server:8000)
   - Fill in registration form
   - Verify email if required

### Setting Up Your Household

1. **Create Household**
   - Go to Settings > Household
   - Click "Create New Household"
   - Enter household name
   - Invite family members by email

2. **Join Household**
   - Accept invitation email
   - Or enter household ID during registration

### Initial Configuration

1. **Profile Setup**
   - Add display name and avatar
   - Set dietary preferences
   - Add allergens to avoid
   - Configure notification preferences

2. **App Preferences**
   - Choose theme (light/dark)
   - Set default location for items
   - Configure unit preferences
   - Enable/disable features

## Inventory Management

### Adding Items

#### Manual Entry
1. Click "Add Item" button
2. Fill in item details:
   - **Name**: Required, descriptive name
   - **Quantity**: Amount you have
   - **Unit**: pieces, kg, liters, etc.
   - **Location**: pantry, fridge, freezer
   - **Category**: Optional, for organization
   - **Expiration Date**: When item expires
   - **Purchase Price**: Optional, for cost tracking

#### Barcode Scanning
1. Click barcode icon
2. Allow camera access
3. Scan product barcode
4. Review auto-filled information
5. Add any missing details
6. Save item

#### Bulk Import
1. Go to Inventory > Import
2. Upload CSV file with items
3. Map columns to fields
4. Review imported items
5. Confirm import

### Managing Items

#### Viewing Items
- **List View**: Default table view
- **Grid View**: Card-based view with images
- **Filter**: By location, category, expiration
- **Sort**: By name, date added, expiration
- **Search**: Find items quickly

#### Editing Items
1. Click on item card
2. Edit any field
3. Save changes
4. Changes sync across all devices

#### Item Actions
- **Use Item**: Log consumption
- **Move Item**: Change location
- **Duplicate**: Create similar item
- **Delete**: Remove item (with confirmation)

## Recipe Management

### Adding Recipes

#### Manual Entry
1. Go to Recipes > Add Recipe
2. Fill in basic information:
   - **Title**: Recipe name
   - **Description**: Brief summary
   - **Prep Time**: Preparation time
   - **Cook Time**: Cooking time
   - **Servings**: Number of servings
   - **Difficulty**: Easy, Medium, Hard
   - **Cuisine Type**: Italian, Mexican, etc.

3. Add ingredients:
   - **Ingredient Name**: What you need
   - **Quantity**: How much
   - **Unit**: cups, tablespoons, etc.
   - **Notes**: Special instructions

4. Add steps:
   - **Step Number**: Order of execution
   - **Instruction**: What to do
   - **Timer**: Optional timer for step
   - **Image**: Photo of this step

5. Add additional info:
   - **Tags**: Dietary tags, meal types
   - **Images**: Recipe photos
   - **Source**: Where recipe came from

#### Import from URL
1. Click "Import Recipe"
2. Enter recipe URL
3. System automatically extracts:
   - Title and description
   - Ingredients and amounts
   - Instructions and steps
   - Images and metadata
4. Review and edit extracted content
5. Save to your collection

#### Import from Image
1. Click "Scan Recipe"
2. Upload or take photo of recipe
3. System uses OCR to extract text
4. Review and correct extracted content
5. Save to your collection

## Meal Planning

### Creating Meal Plans

2. Click "New Plan"
3. Select week to plan
4. Add meals for each day:
   - **Breakfast**: Morning meal
   - **Lunch**: Midday meal
   - **Dinner**: Evening meal
   - **Snacks**: Between meal items
5. Assign recipes to each meal
6. Set number of servings

#### AI-Generated Plans

## Shopping Lists

### Creating Lists

#### Manual Creation
1. Go to Shopping Lists
2. Click "New List"
3. Enter list name
4. Add items manually:
   - **Item Name**: What to buy
   - **Quantity**: How many
   - **Unit**: pounds, liters, etc.
   - **Category**: Produce, dairy, etc.
   - **Notes**: Brand preferences, etc.

#### Auto-Generation
1. From Meal Plan:
   - Click "Generate Shopping List"
   - System adds all needed ingredients
   - Subtracts items you already have
   - Creates organized shopping list

2. From Recipe:
   - Click "Add to Shopping List"
